### How do I run? :eyes:

In project directory, execute the following commands.

1. **Install dependencies:**

    ```sh
    npm install
    ```

2. **Copy environment config:**

    ```sh
    cp .env.development.example .env.development
    ```

3. **Run the development server:**

    ```sh
    npm run dev
    ```

4. **Access the app:**

    ```sh
    http://localhost:5173
    ```
